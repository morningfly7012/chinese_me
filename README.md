# 國寫進步計畫 ✍️

學測國語文寫作能力測驗（國寫）的個人作文分析儀表板：

- **成績總覽**：9 篇作文（知性 5 篇＋情意 4 篇）的得分圖表與走勢
- **能力雷達**：立意取材／結構組織／遣詞造句／審題扣題／深度 五面向分析
- **常犯問題**：跨篇歸納的 6 大弱點，附實例與改進方法
- **逐篇詳解**：每篇的老師批改重點、優缺點、改寫示範與全文謄錄
- **行動計畫**：7 項具體的進步策略

純靜態網站（HTML + CSS + Chart.js），推送到 `main` 後由 GitHub Actions 自動部署至 GitHub Pages。
